# Phase 9: Artifact registry and integrity

## Outcome

One registry classifies, validates, indexes, and relates every current durable artifact kind with per-kind statuses.

## Start condition

Phases 1, 2, and 3 completed and committed.

## Acceptance

- Overview, phase, iterate, spec, review-pass, and draft kinds classify correctly.
- Typed severity replaces string matching.
- Links, counts, status mirrors, and reviewed targets receive deterministic checks.
